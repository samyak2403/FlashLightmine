<p>FlashlightMine is a simple yet elegant Android flashlight application with unique interactive features.</p>

<p><b>Features:</b></p>
<ul>
<li><b>Flashlight Control</b>: Turn the flashlight ON/OFF with a simple button tap</li>
<li><b>Pull Rope Interaction</b>: Drag the lightbulb or line like a pull-cord switch to toggle the flashlight with realistic spring animations</li>
<li><b>Shake Detection</b>: Shake your device to toggle the flashlight - perfect for quick access</li>
<li><b>Interactive Tutorial</b>: A spotlight tutorial guides first-time users on how to use the pull-to-toggle feature</li>
<li><b>Dynamic UI</b>: Background color and light bulb icon change based on flashlight status with beautiful visual feedback</li>
</ul>

<p><b>Privacy Focused:</b></p>
<ul>
<li>No internet permission required</li>
<li>No ads or tracking</li>
<li>Completely open source</li>
</ul>

<p>This app uses the Camera2 API for flashlight control and includes spring physics animations for a premium user experience.</p>
